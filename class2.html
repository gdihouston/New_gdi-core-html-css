What is it?

CSS = Cascading Style Sheets

CSS is a "style sheet language" that lets you style the elements on your page.

CSS works in conjunction with HTML, but is not HTML itself.

What can it do?

All colored text, position, and size

What does it look like?

What is Cascading?

Styles "cascade" down until changed


p{
  color:blue;
  font-family: 'Helvetica';
}
.red{
  color:red;
}
#special{
  font-family: Arial;
}


  <p>Paragraph</p>
  <p class ="green">Paragraph</p>
  <p class ="red">Paragraph</p>
  <p class = "red" id ="special">Paragraph</p>

The CSS Rule

The CSS Rule

                    selector {
  property: value;
}
  • A block of CSS code is a rule.
  • The rule starts with a selector.
  • It has sets of properties and values.
  • A property-value pair is a declaration.

CSS Syntax

Declarations:
  • Contain Property and Value to style HTML element.
  • End with a semicolon (;)
  • Can be grouped together
  • Groups are surrounded by curly brackets

Bonus! What's an HTML Element?

Properties & Values

Properties can have one or more comma separated values.

                    selector {
  property: value;
  property: value;
  property: value;
}
                
p{
  color: white;
  background-color: red;
  font-family: Arial, sans-serif;
}

Selectors:

What To Style

Defines how to pick which HTML elements to style.
Basic selectors include:

  1. Select by Element
  2. Select by ID
  3. Select by Class

Selectors can be nested!

Element

Set the selector value = name of an element

For example, 

                    p {
  property: value;
}

Selects all paragraph elements, while

img {
  property: value;
}

Selects all image elements.

ID

Set the selector value = '#' + ID attribute of an element. The '#' character tells the CSS to search all ID attributes.

For example, 

                    <p id="footer">Copyright 2011</p>
                    #footer {
  property: value;
}

Selects the elements with an id of "footer" in the given HTML

Class

Set the selector value = '.' + class name defined in element. The '.' character tells the CSS to search all class attributes.

                    <p class="warning">Run away!</p>
                    .warning {
  color: red;
}

Selects all elements with a class of "warning" in the given HTML

Nesting

Daisy chaining multiple selectors after each other to define Position of when to apply a style. For example. This selects all em elements that are within a paragraph 

<p>This is <em>important.</em></p>

p em {
  color: yellow;
}

CSS Properties

Many CSS properties have self-explanatory names:

  • background-color
  • font-family
  • font-size
  • color
  • width
  • height

Comprehensive list of all CSS properties

Color

The color property changes the color of the text.


p {
  color: red;
  color: #ff0000;
  color: rgb(255, 0, 0);
}

Color name

Hexadecimal value

RGB value

The 17 standard colors are: aqua, black, blue, fuchsia, gray, grey, green, lime, maroon, navy, olive, purple, red, silver, teal, white, and yellow.

Background-color

The background-color property changes the color of the background.


p {
  background-color: black;
  background-color: #000000;
  background-color: rgb(0,0,0);
}

Font-family

The font-family property deﬁnes which font is used.


p {
  font-family: "Times New Roman";
  font-family: serif;
  font-family: "Arial", sans-serif;
}
  • Specific font name
  • Generic name
  • Comma-separated list

Font-size

The font-size property speciﬁes the size of the font.


p {
  font-size: 12px;
  font-size: 1.5em;
  font-size: 100%;
}

Possible units include:

  • Pixels
  • em
  • Percentage

Fonts (shorthand)


p {
  font-style: italic;
  font-weight: bold;
  font-size: 10px;
  font-family: sans-serif;
}
OR 

p {
  font: italic bold 10px sans-serif;
}
Connecting CSS to HTML

3 ways:

  1. External
  2. Embedded
  3. Inline

Linked


<head>
  <link rel="stylesheet" type="text/css" href="style.css">
</head>
  • Shared resource for several pages.
  • Reduced ﬁle size & bandwidth
  • Easy to maintain in larger projects.
  • Preferred by nerds everywhere!

Embedded


<head>
  <style type="text/css">
    p {
      color: blue;
      font-size: 12px;
    }
  </style>
</head>
  • Inside <head> element.
  • Uses <style> tag.
  • Can only be used in one html file

Inline

<p style="color:red">Some text.</p>
  1. Uses the HTML attribute style.
  2. Difficult to use in large projects
  3. Not preferred.
